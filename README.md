<h1>S2Sh在线答题</h1>
项目介绍：
   这是一个在线答题的网站，可以自动识别单选，多选；同时实现答题之后页面返回使用者答题是否正确的网站，并记录下错误的题目，次数，时间
   使用框架：Struts2+spring+hibernate
   使用数据库：oracle11g
   使用开发工具：myeclipse10
